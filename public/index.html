📋 RÈGLES DU JEU

🎮 MISE EN PLACE

Le premier à créer une salle devient le HOST — c'est le présentateur de la partie. Les autres rejoignent via l'URL du jeu en entrant le code à 5 lettres. Il peut y avoir de 2 à 6 joueurs. Le HOST lance la partie quand tout le monde est prêt.

🎯 DÉROULEMENT D'UN TOUR

1. Le joueur actif (mis en évidence) choisit une case sur le plateau — une catégorie et une valeur de 200 $ à 1 000 $. Seul ce joueur peut cliquer.

2. La question s'affiche pour tout le monde. Le joueur actif tape sa réponse dans le champ prévu et la soumet — tout le monde voit ce qu'il a écrit.

3. Le HOST clique Révéler la bonne réponse. Le groupe débat si la réponse soumise est acceptable.

4. Le HOST tranche : ✅ Bonne ou ❌ Mauvaise. Le tour passe au joueur suivant.

💰 SCORES

✅ Bonne réponse → le joueur gagne la valeur de la case (ex. +600 $).

❌ Mauvaise réponse → la valeur est déduite de son score (ex. −600 $).

⏭ Passer → aucun point gagné ni perdu, la case est retirée.

↩ Annuler → le HOST peut défaire la dernière action si une erreur a été faite.

🏆 FIN DE PARTIE

Quand toutes les cases ont été jouées, le classement final s'affiche avec podium. Le joueur avec le score le plus élevé remporte la partie. En cas d'égalité, les joueurs concernés sont co-champions !

💡 Conseil : jouez avec un appel vocal en parallèle (Discord, FaceTime…) pour débattre des réponses en temps réel !
